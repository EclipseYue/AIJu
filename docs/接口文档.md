# 接口文档

所有接口默认前缀为 `http://localhost:8000`。当前实现为骨架，响应结构已经固定，后续只需替换 service 内部逻辑。

## 1. Health

```http
GET /health
```

响应：

```json
{
  "status": "ok",
  "service": "AIJu"
}
```

## 2. 教材管理

### 上传教材

```http
POST /api/textbooks/upload
Content-Type: multipart/form-data
```

字段：

| name | type | required |
|---|---|---|
| file | file | yes |

响应：

```json
{
  "textbook_id": "book_01",
  "filename": "生理学.pdf",
  "title": "生理学",
  "file_format": "pdf",
  "size_bytes": 32900000,
  "total_pages": null,
  "total_chars": 0,
  "parse_status": "pending",
  "chapters": [],
  "error": null
}
```

### 教材列表

```http
GET /api/textbooks
```

响应：`TextbookSummary[]`

### 解析教材

```http
POST /api/textbooks/{textbook_id}/parse
```

请求：

```json
{
  "force": false,
  "chapter_pattern": "第[一二三四五六七八九十百千万0-9]+章"
}
```

响应：包含 `chapters` 的 `TextbookSummary`。

## 3. 知识图谱

### 构建图谱

```http
POST /api/graph/build
```

请求：

```json
{
  "textbook_ids": ["book_01", "book_02"],
  "max_chapters_per_book": 5
}
```

响应：

```json
{
  "nodes": [
    {
      "id": "node_001",
      "name": "动作电位",
      "definition": "细胞受到刺激后膜电位发生快速可逆倒转。",
      "category": "核心概念",
      "textbook_id": "book_01",
      "textbook_title": "生理学",
      "chapter": "第二章 细胞的基本功能",
      "page": 35,
      "frequency": 1,
      "source_ids": []
    }
  ],
  "edges": [
    {
      "source": "node_001",
      "target": "node_002",
      "relation_type": "prerequisite",
      "description": "理解动作电位需要先掌握静息电位。"
    }
  ]
}
```

### 获取当前图谱

```http
GET /api/graph
```

响应：`KnowledgeGraph`

## 4. 跨教材整合

### 运行整合

```http
POST /api/integration/run
```

请求：

```json
{
  "textbook_ids": ["book_01", "book_02"],
  "compression_target": 0.3
}
```

响应：

```json
{
  "original_chars": 1000000,
  "integrated_chars": 300000,
  "compression_ratio": 0.3,
  "decisions": [
    {
      "decision_id": "merge_001",
      "action": "merge",
      "affected_nodes": ["book01_node_015", "book03_node_032"],
      "result_node": "merged_node_001",
      "reason": "两本教材都讲解炎症概念，保留病理学版本因其描述更完整。",
      "confidence": 0.92
    }
  ],
  "graph": {
    "nodes": [],
    "edges": []
  }
}
```

### 教师反馈

```http
POST /api/integration/feedback
```

请求：

```json
{
  "session_id": "default",
  "decision_id": "merge_001",
  "message": "把抗原和免疫原分开，它们不是同一个概念"
}
```

响应：更新后的 `IntegrationResult`。

## 5. RAG

### 建立索引

```http
POST /api/rag/index
```

请求：

```json
{
  "textbook_ids": ["book_01", "book_02"],
  "chunk_size": 650,
  "overlap_size": 80
}
```

响应：

```json
{
  "indexed_textbooks": 2,
  "chunk_count": 842,
  "status": "ready"
}
```

### 问答

```http
POST /api/rag/query
```

请求：

```json
{
  "question": "炎症的定义是什么？",
  "top_k": 5
}
```

响应：

```json
{
  "answer": "炎症是机体对损伤因子产生的防御性反应。[病理学, 第四章, 第78页]",
  "citations": [
    {
      "textbook": "病理学",
      "chapter": "第四章 炎症",
      "page": 78,
      "relevance_score": 0.92
    }
  ],
  "source_chunks": [
    "炎症(inflammation)是具有血管系统的活体组织对各种损伤因子的刺激..."
  ]
}
```

### 索引状态

```http
GET /api/rag/status
```

响应：`RagStatus`
